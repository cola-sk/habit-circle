# HabitCircle · 习惯圈 — 产品需求文档

> 版本: v0.1 MVP  
> 目标平台: iOS（优先）+ Android（Flutter 跨平台）  
> 定位: 公益性质 · 小圈子 · 学习习惯养成  

---

## 一、产品概述

面向小学生小圈子（学而思班级、兴趣班、邻居群等），通过**完成学习任务获得积分 → 用积分喂养虚拟宠物**的机制，激励孩子坚持学习习惯。家长扫码加入同一个圈子，实时看到所有小朋友的宠物状态，形成正向的同伴压力与互相激励。

---

## 二、核心用户角色

| 角色 | 说明 |
|------|------|
| **孩子** | 完成任务、喂宠物、查看圈子 |
| **家长** | 监督任务完成、确认任务、管理圈子 |

> MVP 阶段：一个账号可以同时扮演两个角色（家长创建账号并帮孩子操作）

---

## 三、核心功能模块

### 3.1 圈子系统（Circle）

- 家长**创建圈子**，获得唯一邀请码（6 位数字）或 **QR 二维码**
- 其他家长通过**扫码 / 输入邀请码**加入圈子
- 一个孩子只属于一个圈子
- 圈子最大人数：MVP 阶段限 20 人
- 圈子有名字（如"三年级二班"、"小书虫2024"）

### 3.2 任务系统（Task）

预设任务模板，家长可自定义开启/关闭：

| 任务名称 | 计分方式 | 默认分值 |
|----------|----------|----------|
| 自主阅读 | 按时长（每15分钟） | 10分 |
| 练琴 | 按时长（每15分钟） | 15分 |
| 英语绘本 | 按时长（每15分钟） | 12分 |
| 课程预习 | 按时长（每15分钟） | 10分 |
| 完成作业 | 完成即得 | 20分 |
| 户外运动 | 按时长（每15分钟） | 8分 |
| 自定义任务 | 家长自定义 | 可设置 |

**任务流程**：
1. 孩子发起"开始任务"（计时器启动）
2. 完成后点击"完成"，自动计算积分
3. 积分进入"待喂食"池，等待喂宠物

### 3.3 宠物系统（Pet）

**宠物状态机**（每日刷新）：

```
状态:  😄 开心 → 😊 正常 → 😕 饿了 → 😢 很饿 → 💀 奄奄一息
条件:  每天喂食 ≥ 80分  /  60-79分  /  30-59分  /  1-29分  /  0分（连续2天）
```

- 每个孩子有一只宠物，初始时选择宠物种类（猫/狗/兔/龙等 3-5 种）
- 宠物有**成长等级**（1-10级），累计总分达到阈值升级
- 升级时宠物外观进化（幼崽 → 少年 → 成年）
- 宠物有名字（孩子命名）
- 积分 = 当天任务总分，当天未使用的积分可以"喂"给宠物

**饥饿机制**：
- 每天 20:00 推送提醒"XX快来喂我！"
- 连续 2 天不喂食，宠物进入"奄奄一息"状态，视觉上明显消瘦
- 连续 7 天不喂食，宠物状态重置（不彻底死亡，保留等级，鼓励重新开始）

### 3.4 圈子广场（Circle Feed）

- 展示圈子内**所有宠物卡片**（网格布局）
- 每张卡片显示：宠物头像（当前状态）、宠物名、主人名、今日状态
- 实时更新（Firestore 实时监听）
- 可以给别人的宠物点"赞/加油"（表情互动）
- 家长视角可以看到孩子今日完成了哪些任务（隐私友好）

### 3.5 账号与认证

- **手机号 + 验证码**注册登录（无需复杂密码，适合家长）
- 首次登录：
  1. 填写孩子姓名 / 昵称
  2. 选择宠物种类并命名
  3. 创建或加入圈子

## 四、MVP 核心页面

```
TabBar 结构:
├── 🏠 首页（我的宠物 + 今日任务）
├── 📋 任务（任务列表 + 计时器）
├── 🌐 圈子（所有宠物广场）
└── 👤 我（个人设置 / 圈子管理）
```

### 页面详情

**首页**：
- 宠物大图展示（带动画：呼吸、互动）
- 宠物状态条（饥饿值、成长值）
- 今日积分 / 喂食按钮
- 今日已完成任务摘要

**任务页**：
- 任务卡片列表
- 进行中任务的计时器
- 今日各任务完成情况

**圈子页**：
- 圈子名称 + 成员数
- 宠物卡片网格（3列）
- 顶部：今日最勤奋的小朋友（积分榜 Top 3）

**我的页**：
- 孩子信息
- 圈子信息 + 邀请码/二维码分享
- 任务自定义开关

|------|------|----------|

---

## 六、非功能性需求

- 数据安全：不收集孩子个人隐私，昵称即可
- 离线支持：任务计时本地完成，联网后同步
- 适龄设计：大字体、明亮色彩、减少文字说明
- 公益性质：MVP 阶段完全免费，无广告

---

## 七、未来迭代方向（Post-MVP）

- 宠物商店（皮肤、装扮道具，积分兑换）
- 家长审核模式（需家长确认任务完成）
- 成就系统（连续30天奖励）
- 多圈子支持
- Web 端家长看板
- 微信小程序版本
